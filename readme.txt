    1. 程序基于rar软件，务必保证安装winRar程序且将其添加到环境变量
    2. 最好在初始页面点击按钮确认是否配置好path环境，不然出错后果自负owo

    初始化界面:
        查看是否配置好path环境，若成功会弹出winrar的界面以及显示弹窗
        更改默认文件解压到的路径
    
    解压:
        从将文件/文件夹 压缩成为一个二次解压的文件
        文件名: 设定好的文件名+校验码+第一层密码(内层)+第二层密码(外层)
        注意:
            密码不能有空格
